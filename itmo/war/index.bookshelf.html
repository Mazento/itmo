Книжная полка

Этот сайт представляет из себя своеобразную "книжную полку" из книг, которые разработчик сайта прочитал или собирается прочитать.

Книги отсортированы по основным жанрам содержащихся книг для простоты ориентирования.

Каждая книга или серия книг имеет краткую информацию об имени автора, название цикла, количество книг в серии и оценку по пятибальной шкале на основе данных с портала goodreads.com

При наведении на обложку раскрывается описание серии. По умолчанию оно скрыто для лаконичности сайта.

Кроме того, на сайте есть страничка с ссылками на полезные электронные ресурсы, относящиеся к тематике данного сайта.

Приятного чтения!

P.S. Информация по реализации:

На этом сайте html страницы получаются напрямую с сервера, однако контент для них (панель навигации, контейнеры для книг) подгружается динамически через Java сервлеты. В частности, контейнеры представляют собой шаблонизированные файлы, в которые подставляются соответствующие данные, распарсенные из дескрипторов книг. В связи с чем, страницы загружаются медленнее, чем если бы весь контент был захардкожен, однако такая реализация обладает лучшей масштабируемостью и меньшим количеством быдлокода.
